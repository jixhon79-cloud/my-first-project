## 2. СТРУКТУРА ПРОДАЮЩЕГО ЛЕНДИНГА

### Идеальная последовательность секций

```
1. HERO              → Кто ты + что делаешь + CTA
2. SOCIAL PROOF BAR  → Цифры / Логотипы клиентов
3. УСЛУГИ            → 3-4 карточки с результатами и ценой
4. ПРОЦЕСС           → 3-4 шага как работаешь
5. КЕЙСЫ             → 3-5 работ с результатами в заголовке
6. ОТЗЫВЫ            → С фото, должностью, компанией
7. О СЕБЕ            → 3-5 предложений + фото
8. FAQ               → 5-7 вопросов закрывают возражения
9. ФИНАЛЬНЫЙ CTA     → Повтор призыва с другой подачей
```

### Топ-5 триггеров доверия для IT-специалиста

1. **Конкретные цифры** — "23 проекта за 3 года", "средний срок — 4 недели"
2. **Прозрачность процесса** — показать инструменты, этапы, как отчитываешься
3. **Логотипы клиентов** — даже 3-5 названий компаний меняют восприятие
4. **Верифицируемые профили** — GitHub с активностью, LinkedIn, Upwork-рейтинг
5. **Снижение риска** — "оплата поэтапно", "первая консультация бесплатно"

### CTA который работает

| CTA | Эффект |
|-----|--------|
| ✅ "Обсудить проект" | Мягко, низкий порог |
| ✅ "Рассказать о задаче" | Клиент — субъект действия |
| ✅ "Получить бесплатную консультацию" | Выгода + нулевой риск |
| ✅ "Написать в Telegram" | Минимальный барьер |
| ❌ "Заказать" | Пугает до принятия решения |
| ❌ "Связаться" | Слишком общее, нет мотивации |

## 3. ТРЕНДЫ ДИЗАЙНА 2025–2026

### Топ-5 визуальных трендов

1. **Bento Grid** — карточки разных размеров как Apple-виджеты, читается за 5 секунд
2. **Dark Mode + неоновый акцент** — тёмный фон (#0A0A0A) + один яркий цвет
3. **Editorial / Brutalist Typography** — гигантские шрифты, текст как визуал
4. **Glassmorphism 2.0** — backdrop-filter точечно, не везде
5. **3D / WebGL в hero** — Three.js, Spline, React Three Fiber

### Анимации которые впечатляют
- **GSAP ScrollTrigger** — отраслевой стандарт для scroll-анимаций
- **Text reveal** — каждое слово появляется последовательно при скролле
- **Magnetic buttons** — кнопка "притягивается" к курсору
- **Custom cursor** — меняет форму/цвет на разных элементах
- **Image reveal on hover** — превью проекта при наведении на ссылку

### Цветовые схемы в тренде

| Схема | Примеры |
|-------|---------|
| Dark + Electric Blue | #0A0A0A + #00D4FF |
| Off-white Minimal | #F5F0EB + #1A1A1A + #FF4D00 |
| Purple Dusk | #0F0A1E + #A855F7 + #FFFFFF |
| Forest / Sage Tech | #0D1F1A + #4ADE80 |
| Monochrome + 1 акцент | #000 + #FFF + #CCFF00 |

### Типографика — шрифты топовых дизайнеров

**Display:** Neue Montreal, PP Neue Machina, Clash Display, Cabinet Grotesk
**Body:** Inter, DM Sans, Satoshi (бесплатный), Plus Jakarta Sans

### Что устарело
- ❌ Карусели и слайдеры с кнопками
- ❌ Агрессивный parallax (тошнит на мобильных)
- ❌ Белый фон + синие кнопки (Bootstrap-стиль)
- ❌ Стоковые иллюстрации
- ❌ Progress bar для навыков (HTML 85%)
- ❌ Typewriter effect на hero

## 4. ЦЕЛЕВАЯ АУДИТОРИЯ

### Портреты заказчиков

**Малый бизнес (50%)**
- Кафе, магазины, салоны, локальные сервисы
- Нужен сайт + базовая автоматизация
- Бюджет: 30 000 – 150 000 ₽
- Боль: "хочу сайт, не понимаю как и сколько стоит"

**Стартапы (25%)**
- MVP, прототип, лендинг для проверки идеи
- Нужна скорость — "вчера"
- Бюджет: 50 000 – 300 000 ₽
- Боль: "нет времени искать команду, нужен результат быстро"

**Агентства и фрилансеры (15%)**
- Аутсорс разработки
- Нужен надёжный подрядчик который не пропадёт

**Личные бренды и эксперты (10%)**
- Коучи, консультанты, блогеры
- Бюджет: 20 000 – 80 000 ₽
- Боль: "хочу выглядеть профессионально, но не технарь"

### Главные возражения
1. "Дорого" → показывать ROI и цену потери без сайта
2. "Долго" → конкретные сроки и этапы
3. "Не доверяю" → кейсы, отзывы, гарантии
4. "Найду дешевле на бирже" → объяснить разницу в качестве

### Где ищут специалистов
- Telegram-чаты и каналы — самый горячий трафик
- По рекомендации — самая высокая конверсия
- ВКонтакте-группы
- FL.ru / Kwork — ценовая конкуренция, не лучший канал

## 5. КОПИРАЙТИНГ

### Топ формулы для H1

```
[Результат] для [Аудитории]
→ "Сайты, которые приносят заявки — для малого бизнеса"

[Глагол] + [Что] + [Без боли]
→ "Строю продукты. Без митингов, без воды, без переделок"

Вайбкодер-стиль:
→ "Строю рабочие продукты с AI — в 10 раз быстрее, чем классическая команда"
→ "Vibe coding + продуктовое мышление = ваш MVP через 2 недели"

Антиформула:
→ "Не агентство. Не джун. Один разработчик — полная ответственность"
```

### Как описывать услуги

| ❌ Плохо | ✅ Хорошо |
|---------|---------|
| "Разработка на React" | "Сайт, который грузится мгновенно" |
| "Telegram-бот" | "Бот, который отвечает клиентам пока вы спите" |
| "AI-агент" | "AI-сотрудник за 1/10 цены менеджера" |
| "UI/UX дизайн" | "Интерфейс, который не нужно объяснять" |
| "Полный цикл разработки" | "От идеи до запуска без третьих лиц" |

### Структура кейса которая убеждает

```
ЗАГОЛОВОК = результат (не название проекта)
↓
ПРОБЛЕМА — боль клиента в 1-2 предложениях
↓
ЧТО СДЕЛАЛ — мои решения (не технологии)
↓
РЕЗУЛЬТАТ — конкретные цифры
↓
ЦИТАТА клиента (если есть)
```

### Слова которые отпугивают
- ❌ "Качественный" / "Уникальный" / "Профессиональный"
- ❌ "Индивидуальный подход"
- ❌ "Работаю на результат"
- ❌ "Берусь за любые задачи"
- ❌ Технический жаргон без перевода (REST API, CI/CD, WebSockets)

## 6. ВАЙБКОДИНГ — ПОЗИЦИОНИРОВАНИЕ

### Что такое вайбкодинг (определение Карпати, февраль 2025)
> "Новый вид кодинга, когда ты полностью отдаёшься вайбу, принимаешь экспоненты и забываешь что код вообще существует"

Суть: **создание продуктов с помощью AI-инструментов** через промпты — без глубокого знания кода.

### Основные инструменты

| Инструмент | Для чего |
|------------|---------|
| **Cursor** | AI-редактор, пишет код по описанию (главный инструмент) |
| **Claude Sonnet** | Архитектура, сложные задачи, код |
| **v0.dev** | UI-компоненты из промпта |
| **Bolt.new** | Полноценные приложения в браузере |
| **Lovable.dev** | SaaS-продукты без кода |
| **Supabase** | Backend/БД за минуты |
| **Vercel** | Деплой одной командой |

### Известные вайбкодеры

- **Pieter Levels (@levelsio)** — PhotoAI, InteriorAI, $200k+/мес
- **Marc Lou** — SaaS за 24-72 часа, курс ShipFast
- **Greg Isenberg** — AI-предпринимательство, YouTube/podcast

### Как объяснить заказчику

> "Я использую современные AI-инструменты которые позволяют создавать сайты и приложения в 5-10 раз быстрее обычной разработки. Вы получаете тот же результат, но быстрее и дешевле."

**Аналогия для заказчика:**
> "Раньше разработчик был и архитектором и строителем. Теперь AI — строитель. Я говорю ему что строить, проверяю качество, корректирую. Стройка идёт в 10 раз быстрее."

### Позиционирование VS конкуренты

| | Агентство | Джун-фрилансер | Вайбкодер |
|--|-----------|----------------|-----------|
| Скорость | Месяцы | Недели | Дни |
| Цена | Высокая | Низкая | Средняя |
| Надёжность | Высокая | Низкая | Зависит |
| MVP | Дорого | Медленно | ✅ Быстро и дёшево |
