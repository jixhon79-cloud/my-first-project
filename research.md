## 2. СТРУКТУРА ПРОДАЮЩЕГО ЛЕНДИНГА

### Идеальная последовательность секций

```
1. HERO          → Кто ты + что делаешь + CTA
2. СОЦИАЛЬНОЕ ДОКАЗАТЕЛЬСТВО → Логотипы клиентов / цифры
3. УСЛУГИ        → Что конкретно делаешь
4. КЕЙСЫ         → 3-5 работ с результатами
5. О СЕБЕ        → Коротко, с личностью
6. ОТЗЫВЫ        → 3-5 реальных отзыва
7. CTA-БЛОК      → Финальный призыв к действию
8. КОНТАКТЫ      → Форма / Telegram / Email
```

### Топ-5 триггеров доверия для IT-специалиста
1. **Конкретные цифры** — "сайт за 7 дней", "увеличил конверсию на 40%"
2. **Логотипы клиентов** — даже 3-5 известных компаний меняют восприятие
3. **Процесс работы** — показать как работаешь снимает страх неизвестности
4. **Реальные отзывы** — со скриншотами переписки, не просто текст
5. **Живое лицо** — фото или видео-приветствие повышает доверие

### CTA который работает
- ✅ "Обсудить проект" — мягко, без обязательств
- ✅ "Получить бесплатную консультацию" — есть ценность
- ✅ "Написать в Telegram" — низкий порог
- ❌ "Заказать" — давит, кажется дорого
- ❌ "Связаться" — слишком общее, нет мотивации

## 3. ТРЕНДЫ ДИЗАЙНА 2025–2026

### Топ-5 визуальных трендов
1. **Bento-сетки** — карточки разного размера как в macOS виджетах
2. **Glassmorphism 2.0** — стекло + blur + неоновые акценты
3. **Cursor-анимации** — кастомный курсор, реагирующий на элементы
4. **Scroll-driven animations** — элементы появляются при скролле (CSS scroll-timeline)
5. **Dark mode by default** — тёмный фон как стандарт для IT-портфолио

### Анимации которые впечатляют
- Текст появляется по буквам при загрузке
- Карточки кейсов разворачиваются при hover
- Параллакс фона при скролле
- Smooth page transitions (GSAP, Framer Motion)
- Magnetic buttons — кнопки "притягиваются" к курсору

### Цветовые схемы в тренде
- **Тёмный + неон** — #0a0a0a + #00ff88 или #7c3aed
- **Молочный + земля** — #f5f0e8 + #2d2d2d (минимализм)
- **Градиент mesh** — плавные цветовые переходы на фоне
- **Монохром + 1 акцент** — всё серое + одна яркая кнопка

### Что устарело
- ❌ Parallax на весь экран (тяжёлый, тошнит на мобильных)
- ❌ Карусели слайдеров
- ❌ Стоковые иллюстрации
- ❌ Светлый фон с синими кнопками
- ❌ "Обо мне" раздел с перечислением навыков списком

### Типографика
- **Display:** Clash Display, Cabinet Grotesk, Syne
- **Body:** Inter, Geist, DM Sans
- Тренд: большие жирные заголовки (font-size 80–120px на десктопе)

## 4. ЦЕЛЕВАЯ АУДИТОРИЯ — КТО ЗАКАЗЫВАЕТ

### Портреты заказчиков

**Малый бизнес (50%)**
- Кафе, магазины, салоны, локальные сервисы
- Нужен сайт + базовая автоматизация
- Бюджет: 30 000 – 150 000 ₽
- Боль: "хочу сайт, но не понимаю как и сколько стоит"

**Стартапы и продуктовые команды (25%)**
- MVP, прототип, лендинг для проверки идеи
- Нужна скорость — "вчера"
- Бюджет: 50 000 – 300 000 ₽
- Боль: "нет времени искать команду, нужен результат быстро"

**Другие фрилансеры и агентства (15%)**
- Аутсорс разработки
- Стабильная нагрузка, не разовый проект
- Боль: "нужен надёжный подрядчик, который не пропадёт"

**Личные бренды и эксперты (10%)**
- Коучи, консультанты, блогеры
- Нужен красивый лендинг + бот/автоматизация
- Бюджет: 20 000 – 80 000 ₽
- Боль: "хочу выглядеть профессионально, но не технарь"

### Главные возражения
1. "Дорого" → показывать ROI, цену потери без сайта
2. "Долго" → конкретные сроки, этапы, дедлайн
3. "Не доверяю" → кейсы, отзывы, гарантии
4. "Найду дешевле на бирже" → объяснить разницу в качестве

### Где ищут специалистов
- Telegram-каналы и чаты (самый горячий трафик)
- По рекомендации (самый конверсионный)
- ВКонтакте-группы
- FL.ru / Kwork (ценовая конкуренция, не лучший канал)
- LinkedIn (для международных клиентов)

## 5. КОПИРАЙТИНГ — ТЕКСТЫ ДЛЯ ЛЕНДИНГА

### Формулы для H1 (главный заголовок)

```
[Результат] для [аудитории] за [срок/условие]
→ "Сайт который продаёт — за 7 дней"

[Глагол действия] + [что] + [для кого]
→ "Создаю AI-продукты для малого бизнеса"

[Проблема] → [Решение]
→ "Устал от подрядчиков которые пропадают? Я не такой."

[Позиционирование] + [суперсила]
→ "Вайбкодер — делаю за 1 неделю то, на что агентства тратят месяц"
```

### Как описывать услуги
- Не "Разработка сайтов" → а "Сайт который приносит заявки с первой недели"
- Не "Telegram-бот" → а "Бот который отвечает клиентам пока вы спите"
- Не "AI-агент" → а "AI-сотрудник который делает работу менеджера за 1/10 цены"

### Блок "О себе" — что работает
```
Короткая история (2-3 предложения):
"Меня зовут [Имя]. Я [кто] который помогает [кому] [что делать].
За [период] создал [цифра] проектов для [типы клиентов]."

+ 1 личная деталь которая запоминается
```

### Слова которые отпугивают
- ❌ "Уникальный" — все так пишут, не верят
- ❌ "Профессиональный" — доказывай делом
- ❌ "Качественно и в срок" — это базовое, не преимущество
- ❌ "Индивидуальный подход" — штамп
- ❌ "Опытный специалист" — без цифр не работает

## 6. ВАЙБКОДИНГ — ПОЗИЦИОНИРОВАНИЕ

### Что такое вайбкодинг
Термин появился в 2025 году (ввёл Андрей Карпатий). Суть:
**создание продуктов с помощью AI-инструментов без глубокого знания кода** — через промпты, Cursor, Claude, v0, Bolt, Lovable.

### Основные инструменты вайбкодера
| Инструмент | Для чего |
|------------|---------|
| **Cursor** | AI-редактор кода, пишет код по описанию |
| **Claude (Anthropic)** | Мышление, архитектура, сложные задачи |
| **v0 (Vercel)** | UI-компоненты из промпта |
| **Bolt.new** | Полноценные приложения из описания |
| **Lovable** | SaaS-продукты без кода |
| **Replit** | Деплой и хостинг |

### Как объяснить вайбкодинг заказчику
> "Я использую современные AI-инструменты которые позволяют создавать сайты и приложения в 5-10 раз быстрее обычной разработки. Вы получаете тот же результат, но быстрее и дешевле."

### Примеры проектов вайбкодеров
- Лендинги и мультистраничные сайты за 1-3 дня
- Telegram-боты с AI за 2-5 дней
- SaaS MVP за 1-2 недели
- Внутренние инструменты для бизнеса
- AI-агенты для автоматизации задач

### Позиционирование на рынке
- **VS агентство:** быстрее в 3-5 раз, дешевле в 2-3 раза
- **VS фрилансер-джун:** качество и скорость выше за счёт AI
- **Уникальное преимущество:** "делаю за неделю то что раньше стоило месяц"

## 7. ВЫВОДЫ ДЛЯ ЛЕНДИНГА

### Главный оффер
```
Создаю [сайты / боты / AI-продукты] для [малого бизнеса / стартапов]
быстро — [срок] — с гарантией результата
```

### Структура лендинга (финальная)
1. **Hero** — оффер + CTA "Обсудить проект в Telegram"
2. **Цифры** — X проектов / X клиентов / X лет
3. **Услуги** — 3 карточки с описанием результата
4. **Кейсы** — 3-4 работы с до/после или результатом
5. **Как работаю** — 3-4 шага, снимает страх
6. **Отзывы** — скриншоты или текст
7. **О себе** — 2-3 предложения + фото
8. **CTA** — "Написать в Telegram / @winterri"

### Ключевое конкурентное преимущество
Вайбкодинг — новая профессия, конкурентов мало, рынок не насыщен.
Кто первым займёт нишу "вайбкодер с портфолио" в Байкальском регионе — тот выиграет.
